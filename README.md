# StreamLine-Movers
Single Page Website for 'StreamLine Movers', packers & movers brand in Canada. 

Tech Stack:

Frontend->

  -Next js = to achieve good seo and quick reloads
  
  -Tailwind css = to style effortlessly
  
Backend->

  -Express js = base of backend
  
  -Nodemailer-sendgrid = to send huge number of emails limitlessly
